# ⚔️ Pokémon Battle Simullator

Uma aplicação interativa de batalha Pokémon desenvolvida para colocar à prova as habilidades de estratégia e lógica. O projeto consome dados reais da PokeAPI e oferece uma experiência imersiva com trilha sonora dinâmica e interface visual moderna.

## 🚀 Funcionalidades

* **Seleção de Equipe:** Interface para pesquisar e escolher Pokémons para dois jogadores diferentes.
* **Atributos em Tempo Real:** Visualização de HP, Ataque e Defesa diretamente nos cards dos jogadores.
* **Trilha Sonora Dinâmica:** A música transita suavemente entre o menu de escolha e o momento da batalha.
* **Cálculo de Vitória:** Lógica que soma os status dos Pokémons para determinar o vencedor da arena.
* **Interface Responsiva:** Design ajustado para funcionar bem tanto em desktops quanto em dispositivos móveis.

## 🛠️ Tecnologias Utilizadas

* **HTML5 & CSS3:** Estrutura e estilização com foco em efeitos de brilho (glow) e layout flexível.
* **JavaScript (ES6+):** Manipulação de DOM, consumo de API assíncrona (Fetch) e gerenciamento de estados de áudio.
* **PokeAPI:** Fonte de dados oficial para as informações e imagens dos Pokémons.
* **Git & GitHub:** Controle de versão e hospedagem do código.

## 🎮 Como Usar

1.  Abra a aplicação e clique em qualquer lugar para ativar a música de introdução.
2.  Clique no botão de busca de cada jogador para selecionar seus Pokémons.
3.  Após escolher 2 Pokémons para cada lado, a música de batalha iniciará automaticamente.
4.  Clique em **Batalhar** para ver o resultado e quem saiu vitorioso.
5.  Use o botão **Jogar Novamente** para resetar a arena.
